# Icons Directory

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create these icons using:

1. **Online Tools**:

   - [Favicon Generator](https://favicon.io/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)

2. **Image Editors**:

   - Photoshop
   - GIMP
   - Figma
   - Canva

3. **Quick Solution**:
   - Create a simple colored square image
   - Export at the required sizes
   - Use a consistent design across all sizes

## Temporary Solution

Until you create proper icons, you can:

1. Create simple colored PNG files at the required sizes
2. Or temporarily comment out the icon references in `manifest.json`

The extension will work without icons, but Chrome will show a default placeholder icon.
